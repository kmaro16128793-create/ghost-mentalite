⚠️ Note technique : revisit-after

La balise meta name="revisit-after" content="7 days" est présente dans le code. À noter : Google ignore cet attribut depuis des années (il n'a jamais été un standard officiel). Il est inoffensif mais sans effet sur le crawl. Les moteurs de recherche modernes décident eux-mêmes de la fréquence de re-crawl.

Fonction unifiée trackAddToCart()

JavaScript — Tracking unifié (5 pixels d'un coup) 
function trackAddToCart() {
  // Google Analytics GA4
  gtag('event', 'add_to_cart', {
    currency: 'EUR', value: 9.90,
    items: [{ item_name: 'Focus Business Membership', price: 9.90 }]
  });
  // Meta Pixel
  fbq('track', 'AddToCart', { value: 9.90, currency: 'EUR' });
  // TikTok Pixel
  ttq.track('AddToCart', { value: 9.90 });
  // Snapchat
  snaptr('track', 'ADD_CART', { price: 9.90 });
  // Microsoft Clarity
  clarity('set', 'AddToCart', 'true');
}

Système d'affiliation via sessionStorage

Un script récupère le paramètre ?ref= dans l'URL et le stocke dans sessionStorage sous la clé focus_ref. Permet de tracker les affiliés sur toute la session sans cookies, en accord avec les règles RGPD.

Stratégie FOMO (Fear Of Missing Out)

Le site utilise 3 techniques combinées : (1) données live simulées sur les cartes hero pour créer un sentiment d'activité intense, (2) Discord phone avec 30 membres en live vocal pour prouver la communauté active, (3) Trustpilot 4.7/5 en double affichage (badge pricing + marquee avis) pour la réassurance.
